明星題材產業，如何分辨出受惠個股

長線存股

  1. ROE成長
  2. EPS成長

短期存股(題材股特徵)

  1. 接到大客戶訂單 (營收 UP)
  2. 報價明顯上揚 (毛利率 UP)
  3. 砍掉賠錢單位 (淨利成長)
  4. 良率提高、報廢率下降 (毛利成長)

題材簡易判斷表

題材 營收 股價
有題材 營收 UP 趨漲
有題材 營收 DOWN 不一定 (有人進去炒作，偏短線。但最好別追)
沒題材 營收 UP 緩漲 (非多數人關注焦點，偏長線)
沒題材 營收 DOWN 趨跌

本益比高為熱門題材股基本特徵，反而要適時追價買進。
熱門題材股通常來得快、去得也快，只適合短線進出。
(尤其是基本面不佳的標的，更不該去碰)

例如， 大立光&玉晶光，後者本益比波動較大，比較適合做追價。(短線操作，要搭配財報表現)

【存股敵8個】第5集！｜3密技推算股票合理價！萬點存股竟能再賺200%？金融黑馬股仍低估？

好股票+好股價＝賺大錢

股票合理價自己算

高登零成長模型

股票股利穩定配發
預期公司營運(股利)穩定，不太會變動發放比率
合理價 = 股利/目標報酬率(可以抓8%左右)
eg. 8049晶采: 2018 現金股利為 2、股票股利為 0，且要求報酬率為 8%，
則合理價 = (2+0)/0.08 = 25

本益比法

成長股，股利發放也增加
合理價 = 預期本益比 * 預期 EPS
eg. 8049 晶采: 以歷史本益比中位數為 9 倍做參考；並用今年前幾期之 EPS 成長幅度來推算全年。

股市反彈還是回升？葛蘭碧八法1秒分辨
反彈是給你逃命的，逃命讓你有資格搶反彈

什麼是反彈

  1. 月線向下
  2. 股價在月線之下
  3. 價漲量縮

    什麼是回升

  4. 月線向上
  5. 股價在月線之上
  6. 價漲量增

    反彈VS回升

葛蘭碧八法

買入時機：

  1. 突破上揚月線：買入
  2. 跌破上揚月線：買入（跌到最低，搶回升）
  3. 回測均線獲得支撐
  4. 月線下彎+乖離過大時：買入（搶反彈）
    需搭配空單避險

賣出時機：

  1. 跌破下降均線：扣抵值20天後沒創高，月線就準備下彎
  2. 棒打出頭鳥：月線往下，突破後隨即下跌
  3. 站不回來：均線下移無法突破
  4. 月線往上，乖離過大
